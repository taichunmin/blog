## About

此網站使用 [VuePress](https://vuepress.vuejs.org/guide/getting-started.html) 來建置，建置教學如下：

```sh
# install globally
yarn global add vuepress # OR npm install -g vuepress

# create a markdown file
echo '# Hello VuePress' > README.md

# start writing
vuepress dev

# build
vuepress build
```
